# Vue.js To-Do List

A simple and clean To-Do List web application built with HTML, CSS, JavaScript, and Vue.js.

## Features
- Add, complete, and delete tasks
- Clear completed or all tasks
- Task counter for pending items

## Tech Stack
- Vue.js
- HTML5
- CSS3
- JavaScript

## Project Setup
Clone the repo and run:


# vue-list

## Project setup
```
npm install
```
